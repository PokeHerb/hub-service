## Poke Herb MSA 물류 서비스
### 허브 서비스
- 물류 서비스의 허브, 허브 간 이동 정보에 대한 서비스를 구현

### 구현 기능
- 허브 관리
- 허브 간 이동 정보 관리 
- 각 허브 간 이동거리, 소요시간 모델을 바탕으로 pgRouting의 다익스트라 알고리즘을 활용하여 최소 비용으로 목적지 허브에 도달하는 경로 계산
- 상품 주문 요청 발생 시
    - 출발지 허브부터 목적지 허브를 거쳐 주문 업체로 도달하는 경로 계산
    - 해당 경로에 대한 예상 소요시간 및 이동거리 계산
    - 허브 간 배송 담당자 배정 요청
    - 업체 배송 담당자 배정 요청
    - 배송 정보 업데이트 요청
    - 상품 재고 수정 요청 (주문된 상품 재고만큼 감소)
    - 주문, 배송 상태 변경 (배송중)
